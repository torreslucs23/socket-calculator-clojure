# Clojure Socket Calculator

This project implements a basic calculator in Clojure using socket communication between a client and server. The client sends arithmetic operations to the server, which performs the calculation and sends the result back to the client.

## Features

- **Socket Communication**: Enables client-server interaction over sockets.
- **Basic Calculator Functions**: Supports operations such as addition, subtraction, multiplication, and division.
- **Client-Server Model**: The client sends requests, and the server processes them and returns results.

## Requirements

- **Clojure**: Ensure you have Clojure installed on your system.
